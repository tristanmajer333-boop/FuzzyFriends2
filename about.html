About Holiday Pet Helpers

We offer a friendly, free holiday care service for small pets. This version uses Firebase Authentication and Cloud Firestore so public reviews, bookings and admin settings are stored in the cloud instead of only in the browser.

📝

1. Check the calendar

Look at the monthly view first so you can see availability, blocked dates and existing bookings.

📦

2. Prepare supplies

Bring food, bedding, toys, housing, medication, cleaning items and written care instructions.

☁️

3. Everything is synced

Bookings and reviews are stored in Firestore, so they can be seen across devices.
